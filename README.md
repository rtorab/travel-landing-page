# Travel Landing Page

A responsive travel landing page built with HTML and CSS.

## Features

- Responsive layout
- Flexbox and CSS Grid
- Custom local font (Inter)
- CSS Variables
- Smooth scrolling
- Hover effects and transitions
- Accessible focus styles

## Technologies

- HTML5
- CSS3
- Flexbox
- CSS Grid
